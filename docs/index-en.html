Open Source license: Affero GPL

On July 27th 2016, the Geneva State Council has chosen to uphold the Affero GPL (version 3) open source license for source code publication, the most up to standards license in the field. The license is written by the Free Software Foundation - a token of ethical accuracy and international renown.

The State of Geneva encourages any IT community to contribute to the source code of this project through the GitHub platform.
